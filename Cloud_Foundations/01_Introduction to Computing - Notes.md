# Introduction to Computing - Notes

## Components of a Computer

A computer consists of:

### Hardware
Physical components of a computer:
- Motherboard
- Central Processing Unit (CPU)
- Memory (RAM)
- Storage Drive
- Network Card

### Software
Programs and applications installed on a computer:
- Operating System (OS)
- Applications

**Examples:**
- OS: Amazon Linux 2, Windows, macOS
- Application: Amazon Shopping Mobile App

---

# Motherboard

The motherboard (system board) is the main circuit board that connects all hardware components.

### Functions
- Holds core computer hardware together
- Connects:
  - CPU
  - Memory (RAM)
  - Storage Drives
  - Network Cards
  - Video Cards
  - Computer Ports

---

# Central Processing Unit (CPU)

The CPU is the **brain of the computer**.

### Responsibilities
- Executes instructions from applications and the OS
- Performs:
  - Arithmetic operations
  - Logical operations
  - Control operations
  - Input/Output (I/O) operations

### CPU Cores
- Multiple cores increase performance.
- Example:
  - Dual-core CPU = 2 processing cores
  - Allows multiple applications to run efficiently

---

# Memory (RAM)

RAM stores data and instructions currently being used by the CPU.

### Characteristics
- Temporary storage
- Data is lost when power is turned off
- Provides fast access to programs and data

### Measurements
- Megabytes (MB)
- Gigabytes (GB)

### Benefits of More RAM
- Better multitasking
- Faster system performance

---

# Storage Drive

Storage drives store:
- Program files
- Application data
- User documents

### Types of Storage

#### Hard Disk Drive (HDD)
- Uses spinning magnetic disks
- Lower cost
- Slower performance

#### Solid State Drive (SSD)
- Uses flash memory
- Faster performance
- More expensive

### Characteristics
- Persistent storage
- Data remains after power is turned off

### Performance Metrics
- MB/s (Megabytes per second)
- IOPS (Input/Output Operations Per Second)

---

# Computer Network

A computer network connects devices to share data and resources.

### Connected Devices
- Computers
- Printers
- Storage Devices
- Smartphones

### Types of Networks

#### Wired Network
- Uses Ethernet cables

#### Wireless Network
- Uses Wi-Fi signals

### Internet
- Largest computer network in the world

---

# Network Interface Card (NIC)

A NIC (Network Adapter) connects a computer to a network.

### Functions
- Enables communication over networks
- Provides internet connectivity

### Types
- Wired NIC (Ethernet)
- Wireless NIC (Wi-Fi)

### Performance Metric
- Gigabits per second (Gbps)

---

# Operating System (OS)

An Operating System manages computer hardware and software.

### Responsibilities
- Coordinates use of:
  - CPU
  - Memory
  - Storage
  - Network resources
- Provides user interface

### User Interfaces
- Command Line Interface (CLI)
- Graphical User Interface (GUI)

### Examples
- Microsoft Windows
- macOS
- Amazon Linux 2
- iOS
- Android

---

# Key Takeaways

1. Applications, computers, and networks power the digital world.
2. An application is a program that performs specific tasks.
3. Hardware refers to the physical components of a computer:
   - CPU
   - Memory
   - Storage Drive
   - Network Card
4. Software refers to programs running on a computer:
   - Operating System
   - Applications
5. The motherboard connects all hardware components.
6. The CPU executes instructions and processes data.
7. RAM provides temporary working storage.
8. Storage drives provide permanent data storage.
9. Networks allow devices to communicate and share resources.
10. The OS manages hardware and software resources.
